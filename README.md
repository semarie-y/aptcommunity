# 아파트 커뮤니티 시설 공급 패턴 분석

## 프로젝트 개요

서울시 신축 아파트 단지의 커뮤니티 시설 공급 패턴을 분석하고,  
인구 구조·세대 구성·주변 상권 데이터를 활용한 수요 간접 예측 분류 모델을 구축합니다.

> **연구 전제**: 시행사는 분양 경쟁력을 위해 지역 수요를 나름대로 반영해 시설을 결정한다고 가정합니다.  
> 따라서 공급 패턴을 수요의 간접 지표로 활용하며, 결과는 '수요 예측'이 아닌 '공급 패턴 예측'으로 해석합니다.

- **과목**: 데이터마이닝
- **작성자**: 양세미 (2024720536)
- **분석 대상**: 서울시 신축 아파트 단지 **964개**

## 핵심 전처리 — 시설 분류 전략

동일 시설이 '피트니스', '휘트니스', 'GX룸', '체력단련실' 등 다양한 표기로 입력되어 있습니다.

## 결측치 처리

단지 데이터(세움터, **법정동** 기준)와 인구·상권 데이터(**행정동** 기준)의  
행정 구역 체계 차이로 결측이 발생하였습니다.

| 처리 단계 | 방법 | 결과 |
|-----------|------|------|
| 1차 — 정확 매칭 | 구+동 이름 LEFT JOIN | 901개 매칭 (86.3%) |
| 2차 — 유사 매칭 | 동 이름 핵심 단어 추출 후 재매칭 ('성수동2가' → '성수') | 828개 매칭 (85.9%) |
| 3차 — 구 평균 대체 | 잔여 결측 → 해당 자치구 평균값으로 보완 | 전체 처리 |
| 4차 — 제거 | 구 평균 산출도 불가한 85개 제거 (강동·성동구 법정동-행정동 불일치) | **최종 964개 확정** |

> **강동·성동구 결측 원인**  
> 세움터의 법정동('성수동2가', '강동' 등)과 행안부의 행정동('성수1·2·3·4가동' 등)이  
> 명칭과 경계 모두 달라 매칭이 불가하였으며, 구 평균 대체도 적용할 수 없어 분석에서 제외하였습니다.

## EDA 주요 발견

- **키즈시설**: 3인이상비율 보유 단지 +2.0%p — 유일하게 유의미한 단변량 차이
- **시니어시설**: 고령비율과 음의 상관(-0.10) — 신축 위주 지역에서 더 많이 공급 (수요-공급 지리적 불일치)
- **공유오피스**: 단변량 상관 +0.03으로 미미하나, 관악구(52%)처럼 지역 수준에서는 패턴 존재
- **아동비율 ↔ 키즈시설**: +0.19 (6개 타겟 중 가장 강한 양의 상관)
