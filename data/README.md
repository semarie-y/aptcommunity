# data

서울시 신축 아파트 커뮤니티 시설 공급 패턴 분석에 사용된 데이터 파일을 모아둔 폴더입니다.
크게 원자료(`raw/`)와 전처리·분석 완료 데이터(`output/`)로 구성되어 있습니다.

```
data/
├── raw/      # 원자료 (인구·세대·복리분양시설 등)
└── output/   # 전처리·분석 완료 데이터 (최종 보고서·Streamlit 앱에서 사용)
```

## 1. data/raw/ — 원자료

Colab 노트북(`notebooks/APT_Community_Analysis.ipynb`)에서 분석 파이프라인의 입력으로 사용되는 원본 데이터입니다.

| 파일 | 설명 | 출처 |
| --- | --- | --- |
| `복지분양시설_서울_통합.csv` | 주택인허가 복리분양시설(커뮤니티 시설) 원자료. 서울시 아파트 단지별 분양 시설 현황 (약 5,441건) | 건축HUB / 세움터 |
| `서울_인구데이터_10년단위_3년치.csv` | 행정동 단위 주민등록 인구통계 (2024년 기준 329개 동). 1인가구·아동·고령 비율 등 인구 구조 변수 산출에 사용 | 행정안전부 주민등록 인구통계 |
| `서울_세대수_디테일전체.csv` | 행정동 단위 세대원수별 세대수 데이터. 1인/2인/3인 이상 가구 비율 산출에 사용 | 행정안전부 주민등록 인구통계 |

## 2. data/output/ — 전처리·분석 완료 데이터

최종 보고서 및 Streamlit 앱(`app.py`)에서 바로 사용하는 전처리 완료 데이터입니다.

### 최종 기준 파일

| 파일 | 설명 | 필수 여부 |
| --- | --- | --- |
| `apt_final_1123.csv` | **최종 분석 데이터셋.** 단지키 기준 서울시 신축 아파트 1,123개 단지의 인구·세대·상권·시설 보유 여부 등 최종 피처가 모두 포함된 파일 | 필수 |
| `step7_target_summary.csv` | 시설 유형별 보유 단지 수와 보유율 요약 (키즈시설, 시니어시설, 운동시설 등 6개 타겟) | 권장 |
| `step9_best_model_summary.csv` | 시설 유형별 최종 채택 모델(Random Forest / Logistic Regression)과 보유 클래스 F1, Macro F1 등 성능 요약 | 권장 |
| `step10_shap_top3_summary.csv` | 시설 유형별 SHAP 분석 기준 영향력 Top 3 변수 요약 | 권장 |
| `step11_hypothesis_final_conclusion.csv` | Q1~Q4 가설에 대한 최종 판정(기각/지지/부분 지지) 및 핵심 해석 정리표 | 권장 |

### Legacy 파일

기존 중간분석 및 Streamlit 초기 버전에서 사용한 `apt_final_v3_biz.csv`, `apt_final_v2_jumin.csv`, `복지분양시설_단지단위.csv` 등은 필요 시 유지할 수 있으나, **최종 보고서 기준 데이터는 `apt_final_1123.csv`**입니다.

### 업로드하지 않는 파일

원본 소상공인 상권정보 파일은 용량이 크므로 GitHub에 올리지 않습니다. Colab에서 직접 업로드하여 재현합니다.

## 3. 데이터 파이프라인 흐름

```
data/raw/ (원자료)
   │  Colab 노트북: notebooks/APT_Community_Analysis.ipynb (1~12단계)
   ▼
data/output/ (전처리·분석 완료 데이터)
   │
   ├─▶ app.py (Streamlit 대시보드)
   └─▶ reports/최종보고서_양세미_v4.docx (최종 보고서)
```
