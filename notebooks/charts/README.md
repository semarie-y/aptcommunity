## 최종 보고서용 필수 차트

| 파일 | 설명 |
|---|---|
| `chart1_facility_rate.png` | 시설 유형별 보유율 |
| `chart2_feature_dist.png` | 인구·세대 변수 분포 |
| `chart3_corr_heatmap.png` | 피처 × 시설 타겟 상관관계 히트맵 |
| `chart4_facility_comparison.png` | 시설 보유 여부별 인구·세대 특성 차이 |
| `chart5_gu_heatmap.png` | 자치구별 시설 보급률 히트맵 |
| `chart6_hypothesis_scatter.png` | Q1~Q4 핵심 가설 예비검증 |
| `chart8_shap_importance.png` | 시설별 SHAP 변수 중요도 |
| `chart9_q4_direction.png` | Q4 외부 상권 방향 분석 |
| `chart10_hypothesis_final_conclusion.png` | Q1~Q4 최종 결론표 |
